Các lệnh cho Ubuntu:
- mkdir "tên folder" : tạo folder
- cd "~" / ".." / "tên folder" : "di chuyển đến folder mặc định của Ububtu" / "Lui ngược về folder cha" / "di chuyển đến folder đó"
- wsl -l -v : Kiểm tra các hệ điều hành linux đang có trên máy "*" ở chỗ nào thì có nghĩa WSL đang dùng nó làm hệ điều hành chính 
- wsl --set-default Ubuntu : set hệ điều hành chính mà WSL nhận là "Ubuntu" thay vì là Docker(mặc định)

* venv: là tạo một môi trường ảo riêng biệt mới để k phải phụ thuộc vào môi trường mặc định chung
- sudo apt install python3-venv : cài đặt gói python3-venv chung
- rm -rf venv: Xóa thư mục venv cũ lỗi nếu có
- python3 -m venv venv : Tạo lại venv
- source venv/bin/activate : Kích hoạt venv

Sửa lỗi python k tìm thấy thư viện
Ctrl + Shift + P -> Python: Select Interpreter -> sẽ có ds hiện ra chọn cái có đường dẫn là môi trường mình đang chạy

- Build Image để đưa lên docker(vd: innomedia-media-engine) : docker build -t innomedia-media-engine . (dấu . báo cho Docker biết là hãy dùng file Dockerfile ở thư mục hiện tại).
- Sau khi build Image xong, ta Run : docker run -d -p 8001:8001 --name media-engine-container innomedia-media-engine
- Kiểm tra Log của một container để xem quá trình làm việc của nó : docker logs media-engine-container / docker logs -f media-engine-container (nếu muốn xem theo thời gian thực)
- Nhớ sau khi cập nhật lại Image của docker ta buộc phải xóa container cũ đi : docker rm -f media-engine-container (tham số -f là force: buộc container đó phải dừng lại và xóa nó.)
